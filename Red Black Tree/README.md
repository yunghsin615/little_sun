Red Black Tree
-
以Binary Search Tree為基礎的演算法<br>
差異是在他需要**平衡**，在insert和delete的時候，要讓整棵樹平衡<br>
首先有一些限制<br>
一:每個節點不是紅或是黑<br>
二:ROOT必須是黑<br>
三:若節點為紅，其子節點必為黑<br>
四:若節點為嘿，其子節點可為紅或黑<br>
五:每個空節點都是黑<br>
六:從ROOT到leaf的每條路徑，必包含相同數目的黑色節點<br>
